# Khmer DUB Studio

## Project Overview
Khmer DUB Studio is an AI-assisted Cambodian dubbing and subtitle production platform.

The system combines:
- subtitle editing
- speaker management
- AI voice generation
- timeline synchronization
- dubbing workflow automation
- video preview
- audio export

## Main Workflow
1. Upload video
2. Load subtitles (SRT)
3. Assign speakers
4. Generate Khmer dubbing audio
5. Sync clips on timeline
6. Preview video with subtitles
7. Export final dubbed audio/video

## Translation Style Rules
- Use natural conversational Khmer
- Avoid overly formal Khmer
- Preserve emotional tone
- Preserve character personality
- Optimize for dubbing rhythm
- Prefer short spoken phrases

## Subtitle Rules
- Maximum 2 lines
- Maximum 42 characters per line
- Keep mobile readability
- Avoid awkward line breaks

## Audio Standards
- AAC stereo export
- Target loudness: -14 LUFS
- Dialogue clarity priority
